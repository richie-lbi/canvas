EaselJS > Intro

  1. EaselJS - it's Flash for HTML5!
  2. Size: 62k
  3. Support: IE9 +
  4. EaselJS is Javascript so supports usual stuff like Objects and Arrays etc as well as familiar syntax
  5. Uses createjs namespace
  6. Classes of note: DisplayObject, Stage, Container, Bitmap (Draws an image, video or canvas to the canvas according to its display properties), BitmapAnimation, Shape, Graphics, Text, DOMElement, Filter
  7. DisplayObject : Abstract base class for all display elements in EaselJS. Exposes all of the display properties (ex. x, y, rotation, scaleX, scaleY, alpha, shadow, etc) that are common to all display objects.
  8. Support of touch devices is built in
Comes as part of createJS suite
