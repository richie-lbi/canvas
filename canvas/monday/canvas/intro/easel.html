EaselJS > Intro

  1. http://www.createjs.com/#!/EaselJS - it's Flash for HTML5!
  2. Uses Canvas so IE9 and above
  3. EaselJS is Javascript so supports for Objects and Arrays etc
  4. Uses createjs namespace
  5. DisplayObject : Abstract base class for all display elements in EaselJS. Exposes all of the display properties (ex. x, y, rotation, scaleX, scaleY, alpha, shadow, etc) that are common to all display objects.
