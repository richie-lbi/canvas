KineticJS > Intro

  1. KineticJS - a canvas JavaScript library that extends the 2d context
  2. Size: 88k
  3. Support: IE9+
  4. It's Javascript - with an Object Oriented API
  5. Guiding principles: Speed, Scale (lots of nodes), Flexibility, Familiar API (for devs with HTML, CSS, JS, and jQuery backgrounds) Extensibility
  6. The stage, layers, groups, and shapes are virtual nodes, similar to DOM
  7. Uses Kinetic namespace
  8. Desktop and mobile applications - extensive event model
