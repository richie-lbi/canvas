PaperJS > General Stuff

  1. PaperJS Swiss Army Knife of vector graphics scripting.
  2. Size: 56k
  3. Support: IE9 +
  4. Uses either javascript or it's own flavour of javascript - PAPERSCRIPT
  5. Has own DOM model - often called a 'scene graph'. A node based way of working with its elements
  6. Based on Scriptographer for Adobe Illustrator
  7. Uses Paths, Points, Vectors and Segments
