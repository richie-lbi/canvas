EaselJS > Drawing

  1. Create a stage - stage becomes canvas element
  2. Draw a shape
  3. Add to stage
  4. Don't forgt to update stage
  5. Tiny Drawing API for vector graphic drawing
Drawing the 'L' 
var stage = new createjs.Stage("easel-canvas");

var shape1 = new createjs.Shape();
shape1.graphics.beginFill("red").dr(-20, -20, 40, 40);

stage.addChild(shape1);
stage.update();
